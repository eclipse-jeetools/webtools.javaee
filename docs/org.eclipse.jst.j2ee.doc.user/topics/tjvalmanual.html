Manually validating code

When you run a manual validation, all resources in the selected project are validated with the validators that are selected on the Validation page of the project Properties dialog.

To manually invoke an immediate code validation:

  1. Select the project that you want to validate.
  2. Right-click the project and click Run Validation. If no validators were selected, a message box provides information about how to select a validator.
Validation of the project is performed using the selected validators for the project.
